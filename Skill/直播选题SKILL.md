# 直播选题分析 Skill

## 使用场景

当我准备做一场直播，但还不确定主题、目标人群、内容框架和转化方式时，使用这个 Skill。

## 输入信息

用户需要提供：

1. 直播主题初稿
2. 目标人群
3. 想卖的产品或服务
4. 当前流量来源
5. 希望直播达成的结果

## 分析步骤

### 第一步：判断主题吸引力

判断这个主题是否有明确痛点、是否容易被用户理解、是否有点击欲望。

### 第二步：判断目标人群

分析这个主题最适合吸引哪类人，以及这类人的核心需求是什么。

### 第三步：拆内容框架

输出 3-4 个直播核心模块，每个模块要解决一个具体问题。

### 第四步：设计转化路径

判断直播最后应该如何自然引导用户加入社群、购买课程或咨询产品。

## 输出格式

请按照下面格式输出：

1. 直播主题建议
2. 目标人群分析
3. 直播内容框架
4. 转化设计
5. 风险提醒
6. 优化建议
