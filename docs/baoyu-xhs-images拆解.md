# baoyu-xhs-images 拆解

## 这个 Skill 是什么？

这是一个用于生成小红书/社媒图片卡片的 AI Skill。

## 它解决什么问题？

把一段复杂内容拆成适合社交媒体传播的图片卡片系列。

## 适合什么场景？

- 小红书图文
- 小红书种草
- 微信图文
- 图片卡片
- 社媒信息卡片

## 它的核心输入是什么？

1. 原始内容
2. 生成几张卡片
3. 使用什么视觉风格
4. 使用什么布局
5. 使用什么配色

## 它的核心输出是什么？

一组图片卡片。

## 我从这个 Skill 学到什么？

一个好的 AI Skill 不是简单写提示词，而是要写清楚：

1. 什么时候使用
2. 需要用户提供什么
3. 如果信息不够，应该追问什么
4. 生成结果有什么标准
5. 如何处理工具选择
6. 如何检查最终结果
